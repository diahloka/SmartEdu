# SmartEdu - Tutoring Management System

---

### 📝 Project Overview
SmartEdu is a desktop-based education management system designed for tutoring institutions to help administrators manage students, tutors, courses, classes, schedules, and payment records in a centralized platform. 
Developed as a collaborative group project by a 5-person team, the application follows an **MVC-like architecture** to separate business logic, user interface, and data access layers.

---

### 🛠️ Tech Stack & Tools

*   **Language & GUI Framework:** Java (Java Swing)
*   **Database:** MySQL
*   **Version Control:** Git & GitHub
*   **Development Environment:** IntelliJ IDEA
